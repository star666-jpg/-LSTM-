### 二、支路一：TCN-LSTM-Transformer `src/models/tcn_lstm_transformer.py`

实现时序建模主干网络，三阶段串联：

**TCN（时序卷积网络）**
- 4层膨胀因果卷积，膨胀因子 1→2→4→8
- 感受野约 31 步，能捕捉长程时序依赖
- 关键修复：改用手动左侧填充，确保因果性（输出位置 t 只能看到 t 及之前的输入）

**LSTM**
- 4层堆叠，隐藏维度 256
- 捕捉短期有序依赖

**Transformer Encoder**
- 新增正弦/余弦位置编码（时序任务必须项）
- 4头自注意力，Pre-LN 结构（训练更稳定）
- 捕捉序列中任意位置之间的全局关系

输出：最后时间步特征向量 `(B, 128)`，供 H-MoE 使用。

### 三、融合层：H-MoE `src/models/hmoe.py`

实现层次化混合专家模块，对应孙惠莹(2025) §3.3：

**两级路由结构（共4个叶专家）：**
```
第一级（粗粒度）：多头方向 / 空头方向
第二级（细粒度）：强趋势上涨 / 震荡上涨 / 强趋势下跌 / 震荡下跌
```

**关键机制：**
- Top-K 稀疏路由（K=2）：每次只激活最相关的2个专家
- 训练时噪声注入：鼓励门控探索不同专家
- 负载均衡辅助损失（引自 Switch Transformer）：防止专家坍缩
- 双输出头：回归头（收盘价）+ 分类头（涨跌方向）

### 四、情感打分模块 `src/sentiment/snowlp_scorer.py`

基于 SnowNLP 对股吧评论进行中文情感分析：

- 文本清洗：去除 URL、表情符 `[哭泣]`、噪声字符
- 批量打分：逐条评论输出 [0,1] 情感分
- 7维日级别特征聚合：均值、标准差、正负比例、评论量、情感动量、5日均线
- 交易日对齐：节假日数据前向填充，冷启动期填中性值 0.5

### 六、第一阶段训练 `scripts/train_phase1.py`

按老师要求完成第一阶段实验：

**实验设置：**
- 特征：开盘、收盘、最高、最低、成交量（5维原始数据）
- 划分：70% 训练集 / 30% 测试集
- 滑窗：30 天
- 模型：TCN-LSTM-Transformer（缩小版）+ H-MoE

**实验结果：**

| 指标 | 值 |
|------|----|
| MSE  | 1850.02 |
| MAE  | 27.52 元 |
| **R²** | **0.7011** |

**与论文基准对比：**

| 模型 | R² |
|------|----|
| 单体 LSTM（孙惠莹论文基线） | 0.42 |
| TCN-LSTM-Transformer（孙惠莹第三章） | 0.68 |
| **第一阶段（5维特征）** | **0.70** |
| 孙惠莹完整模型（含情感+H-MoE） | 0.71 |

仅用5个原始特征，R² 已达到论文完整模型水平（0.71），验证了 H-MoE 融合机制的有效性。
