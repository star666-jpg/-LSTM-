TCN 时域卷积网络

Temporal Convolutional Network — 膨胀因果卷积捕捉多尺度局部时序模式

核心思想

因果性（Causal）

位置 t 的输出只依赖于输入序列中 t 及之前的时间步，不会"看到未来"。

实现方法：左侧填充 (kernel-1)×dilation 个零，右侧不填充。

未来数据泄露会导致测试集虚高的预测精度， 因果卷积是金融预测的硬性要求。

膨胀卷积（Dilated）

膨胀率按 1→2→4→8 指数增长，卷积核感受野随层数指数级扩大，用少量参数覆盖长时序。

4层堆叠后感受野：

RF = 2 × (kernel-1) × (2⁴ - 1) + 1
kernel=3 → RF = 2×2×15+1 = 61 步

TCN 残差块结构

为什么是 TCN 而不是纯 CNN？

普通 CNN 的问题

padding=same 时，右侧会补零，导致时刻 t 的输出看到了 未来的数据，造成信息泄漏。

TCN 的解决方案

使用 因果填充（左侧填充），配合 weight_norm 稳定训练。残差连接解决深层梯度问题。

LSTM 长短期记忆网络

Long Short-Term Memory — 三门控制解决 RNN 梯度消失问题

为什么需要 LSTM？

普通 RNN 的问题

隐藏状态 ht = tanh(Wht-1 + Uxt + b)

反向传播时梯度连乘 W^T，长序列导致梯度爆炸或消失，无法学到10步以上的依赖。

LSTM 的解决方案

引入细胞状态 Ct（Cell State）作为"高速公路"，信息可以几乎无损地流经很长的时间步。

三个门学习"何时遗忘、何时记住、何时输出"。

🗑️
遗忘门 (Forget Gate)
ft = σ(Wf·[ht-1, xt] + bf)
输出 [0,1]，0=完全遗忘，1=完全保留。
决定细胞状态中哪些旧信息要丢弃。
股价示例：遇到季报发布，遗忘门会让模型"忘掉"之前的价格趋势。
σ输出示意 ≈ 0.65（保留65%旧信息）
✍️
输入门 (Input Gate)
it = σ(Wi·[ht-1, xt] + bi)
C̃t = tanh(Wc·[ht-1, xt] + bc)
两步：① sigmoid 决定写入多少；② tanh 产生候选值 [-1,1]。
股价示例：RSI超卖信号出现时，输入门会写入"超卖状态"信息。
it·C̃t = 写入新信息量
📤
输出门 (Output Gate)
ot = σ(Wo·[ht-1, xt] + bo)
ht = ot ⊙ tanh(Ct)
决定从细胞状态中读取哪些信息作为当前输出 ht。
股价示例：预测时，输出门决定将长期趋势中的哪部分传递给下一层。
ht = ot⊙tanh(Ct)
细胞状态更新： Ct = ft ⊙ Ct-1 + it ⊙ C̃t
（遗忘旧信息 + 写入新信息，⊙ = 逐元素相乘）

Transformer 编码器

Self-Attention 机制捕捉序列中任意时间步之间的全局关系

为什么在 LSTM 之后还需要 Transformer？

LSTM 的局限

LSTM 虽然有记忆，但信息是顺序压缩的：t=1 的信息经过64步传递后大量衰减。

例：今天的价格与30天前的某个关键日期的关系，LSTM 很难直接捕捉。

Transformer 的优势

Self-Attention 可以让 t=64 直接"看到" t=1，没有距离衰减。

4个注意力头各自专注不同的时间模式：短期、中期、长期、波动。

下面矩阵的每个格子代表"第 i 个时间步关注第 j 个时间步的程度"，颜色越亮权重越高（动画随机模拟）：

注意力矩阵 (6×6示意)
列 = 被关注的时间步

计算过程（缩放点积注意力）

① 线性投影
Q = X·WQ, K = X·WK, V = X·WV
X: (B, T, 256) → Q,K,V: 各 (B, T, 64) per head
② 注意力权重
Attention(Q,K,V) = softmax(QKT / √dk) · V
√dk = √64 = 8 （防止点积过大梯度消失）
③ 多头拼接
MultiHead = Concat(head₁,...,head₄) · WO
4头 × 64维 = 256维，保持维度不变

位置编码（Positional Encoding）

Transformer 本身无顺序感知，需手动注入位置信息：

PE(pos, 2i) = sin(pos / 100002i/d)
PE(pos, 2i+1) = cos(pos / 100002i/d)

使用 Pre-LN（LayerNorm 在 Attention 之前）：训练更稳定，收敛更快，避免后期 loss spike。

① 因果卷积（TCN 核心）

class _CausalConv1d(nn.Module):
    """
    因果一维卷积：t时刻输出只能看到 t 及之前的输入。
    在左侧填充 (kernel-1)*dilation 个零，右侧不补。
    """
    def __init__(self, in_ch, out_ch, kernel, dilation):
        super().__init__()
        self.left_pad = (kernel - 1) * dilation   # 保证因果性的关键
        self.conv = weight_norm(
            nn.Conv1d(in_ch, out_ch, kernel, dilation=dilation, padding=0)
        )

    def forward(self, x):          # x: (B, C, T)
        x = F.pad(x, (self.left_pad, 0))  # 只在左侧补零
        return self.conv(x)

② TCN 残差块

class _TCNBlock(nn.Module):
    def __init__(self, in_ch, out_ch, kernel, dilation, dropout):
        super().__init__()
        self.net = nn.Sequential(
            _CausalConv1d(in_ch,  out_ch, kernel, dilation),
            nn.ReLU(), nn.Dropout(dropout),
            _CausalConv1d(out_ch, out_ch, kernel, dilation),
            nn.ReLU(), nn.Dropout(dropout),
        )
        # 维度不同时用 1×1 卷积对齐残差
        self.downsample = nn.Conv1d(in_ch, out_ch, 1) if in_ch != out_ch else None
        self.relu = nn.ReLU()

    def forward(self, x):
        residual = x if self.downsample is None else self.downsample(x)
        return self.relu(self.net(x) + residual)  # 残差连接

③ TCN 4层膨胀堆叠

class TCN(nn.Module):
    def __init__(self, in_features, hidden=64, n_levels=4, kernel=3, dropout=0.3):
        super().__init__()
        layers = []
        for i in range(n_levels):
            in_ch = in_features if i == 0 else hidden
            layers.append(_TCNBlock(in_ch, hidden, kernel,
                                     dilation=2 ** i,  # 1,2,4,8
                                     dropout=dropout))
        self.network = nn.Sequential(*layers)

    def forward(self, x):    # x: (B, T, F)
        # Conv1d 需要 (B,C,T)，前后转置
        return self.network(x.permute(0, 2, 1)).permute(0, 2, 1)

④ 主模型前向传播

def forward(self, x):
    """
    x: (B, T, in_features)  ← 技术指标+情感特征时序窗口
    """
    # Step 1: TCN — 多尺度局部特征提取
    x = self.tcn(x)                         # (B,T,21) → (B,T,64)

    # Step 2: LSTM — 顺序时间依赖建模（4层，hidden=256）
    x, _ = self.lstm(x)                     # (B,T,64) → (B,T,256)

    # Step 3: Transformer — 全局自注意力（任意时间步间关系）
    x = self.pos_enc(x)                     # 注入正弦位置编码
    x = self.transformer(x)                 # (B,T,256) → (B,T,256)

    # Step 4: 取最后时间步，归一化，线性投影到128维
    feat = self.proj(self.norm(x[:, -1, :])) # → (B,128)
    return feat                             # 供 H-MoE 融合
