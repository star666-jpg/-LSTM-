**符号约定（全文通用）**

| 符号 | 含义 |
|------|------|
| $x_t \in \mathbb{R}^F$ | $t$ 时刻的输入特征向量，$F$ 为特征维度 |

| $[\,;\,]$ 或 $[\,,]$ | 向量拼接 |
| $B,T,F$ | Batch大小、时间步数、特征维度 |

## 一、为什么要把三个模型串在一起？

先说结论：**三个模块各有盲区，串联起来互相补位。**

| 模块 | 擅长 | 做不好的事 |
|------|------|-----------|
| TCN | 识别局部波形（K线形态、短期震荡） | 无法直接联系"30天前"和"今天" |
| LSTM | 记住事件顺序（前几天的走势影响今天） | 信息传远了会衰减，类似"传话游戏" |
| Transformer | 让任意两个时间步直接"对话" | 自己不知道先后顺序，需要外部注入 |

三者串联后的分工：
1. **TCN 先看"局部形态"**——把原始特征变成更有意义的局部模式
2. **LSTM 再理"时间顺序"**——把这些模式按时间线串起来形成记忆
3. **Transformer 最后做"全局梳理"**——让30天前的某个关键信号直接影响今天的判断

## 二、数据是怎么流进来的？

### 2.1 第一阶段：5维原始数据

最简单的输入，就是日度行情的五列：

```
open（开盘）  close（收盘）  high（最高）  low（最低）  volume（成交量）
```

用一个**滑动窗口**把连续 30 天打包成一个样本，预测第 31 天的收盘价。

```
窗口大小 30 天 × 5 维 = 每个训练样本的形状 (30, 5)
```

### 2.2 扩展到技术指标：15 维

原始价格信息太"裸"，很多交易信号藏在派生指标里。用 `ta` 库自动计算常见指标，保留15列：

| 类别 | 指标 | 直觉含义 |
|------|------|---------|
| 趋势 | SMA、EMA、MACD、CCI、ADX | "现在是上升趋势还是下跌趋势？" |
| 动量 | RSI、Stoch、ROC | "涨/跌的速度在加快还是减慢？" |
| 波动率 | 布林带上下轨、ATR | "现在波动大不大？" |
| 成交量 | MFI、OBV | "资金在流入还是流出？" |

### 2.3 加入情感特征：+7 维

从股吧爬取贵州茅台的评论，用 **SnowNLP** 给每条评论打 [0,1] 分（越高越正向），然后按交易日聚合成7个数字：

| 特征 | 捕捉什么 |
|------|---------|
| `sentiment_mean` | 今天整体情绪偏正向还是负向 |
| `sentiment_std` | 多空双方分歧大不大 |
| `positive_ratio` | 看涨的人占多少 |
| `negative_ratio` | 看跌的人占多少 |
| `comment_count` | 今天讨论热度（取log，避免极端值） |
| `sentiment_momentum` | 情绪是在好转还是在恶化（与昨天的差值） |
| `sentiment_ma5` | 5天情绪均值，过滤单日噪声 |

**最终输入张量**：窗口 64 天 × (15 技术指标 + 7 情感) = **(64, 21)**

### 数据预处理关键公式

**MinMax 归一化**（在训练集上 fit，防止数据泄露）：

$$\hat{x} = \frac{x - x_{\min}}{x_{\max} - x_{\min}} \in [0, 1]$$

**滑动窗口切分**：

$$\text{样本}_i = \bigl(X[i : i+T],\; y[i+T]\bigr), \quad i = 0, 1, \ldots, N-T-1$$

- $T=64$：窗口大小，$X[i:i+T] \in \mathbb{R}^{64 \times 21}$
- $y[i+T]$：第 $i+T$ 天的收盘价（第4列），为回归目标
- 数据集划分比例 $6:1.5:2.5$（训练/验证/测试）

## 三、TCN：用"膨胀卷积"读时序

### 直觉类比

普通卷积就像你读文章时每次只看相邻3个字（核大小=3）。  
膨胀卷积像戴了一副"变焦眼镜"——

- 第1层：看字1、2、3（步幅=1）
- 第2层：看字1、3、5（跳1个，感受野翻倍）
- 第3层：看字1、5、9（跳3个）
- 第4层：看字1、9、17（跳7个）

4层叠加后，**最后一个位置能"看到"前31步**的信息，而参数量没有显著增加。

### 因果性为什么重要？

普通卷积做两侧填充（padding=same），会让 t 时刻看到 t+1、t+2 的数据——**等于预测时偷看了答案**。

TCN 只在左侧填充（只看过去），输出严格基于历史，不泄漏未来。

### 核心公式

**膨胀因果卷积（单层输出）**

$$y_t = \sum_{k=0}^{K-1} w_k \cdot x_{t - d \cdot k}, \quad t \in \{0,1,\ldots,T-1\}$$

- $K$：卷积核大小（本项目 $K=3$）
- $d$：膨胀率，第 $i$ 层取 $d = 2^{i-1}$（即 $1,2,4,8$）
- $x_{t-d\cdot k}$：只访问 $t$ **之前**的位置，保证因果性

**感受野（Receptive Field）**

$$\text{RF} = 1 + 2(K-1)\sum_{i=0}^{L-1}2^i = 1 + 2(K-1)(2^L - 1)$$

本项目 $K=3,\,L=4$：$\text{RF} = 1 + 2 \times 2 \times 15 = \mathbf{61}$ 步。

> 实际代码用的是"左侧填充 $(K-1)\times d$ 个零"来实现，代码中对应：
> ```python
> self.left_pad = (kernel - 1) * dilation
> x = F.pad(x, (self.left_pad, 0))   # 只在左侧补零
> ```

### 残差块公式

设第 $i$ 层输入为 $\mathbf{u}$，经过两层因果卷积 + 激活后得到 $\mathcal{F}(\mathbf{u})$：

$$\text{Block}_i(\mathbf{u}) = \text{ReLU}\!\left(\mathcal{F}(\mathbf{u}) + \mathbf{u}'\right)$$

其中 $\mathbf{u}' = \mathbf{u}$（若输入输出通道相同）或 $\mathbf{u}' = \text{Conv}_{1\times1}(\mathbf{u})$（通道不同时对齐维度）。

每层 TCN 除了主路（卷积）还有一条"高速公路"（shortcut）直接把输入加到输出。好处：

- 梯度可以通过加法直接传回浅层，不消失
- 即使主路学废了，网络至少还有恒等映射兜底

**TCN 输出**：每个时间步从21维变成64维的"局部特征表示"，形状 $(B, T, 64)$。

## 四、LSTM：三把锁控制记忆

### 为什么不用普通 RNN？

普通 RNN 靠一个隐藏状态 $h_t$ 传递记忆，每步都乘一个权重矩阵。  
100步之后，梯度连乘了100次，不是爆炸就是消失——**远的事情根本记不住**。

### LSTM 的解决方案：细胞状态 + 三个门

把"记忆"分成两条轨道：
- **细胞状态 $C_t$**：类似"档案柜"，信息可以几乎不损耗地流很远
- **隐藏状态 $h_t$**：类似"工作记忆"，用来传给下一层

三个门控制信息的"存取删"：

#### 🗑️ 遗忘门（Forget Gate）
> "旧档案里哪些该丢掉？"

$$\boldsymbol{f}_t = \sigma\!\left(W_f \cdot [h_{t-1};\, x_t] + b_f\right)$$

- 输入：上一步隐藏状态 $h_{t-1}$ 与当前输入 $x_t$ 拼接
- 输出：$\boldsymbol{f}_t \in (0,1)^H$，逐维控制"保留多少旧信息"
- 接近 0 → 彻底遗忘；接近 1 → 完整保留

**股市例子**：季报公布后，模型通过遗忘门"清空"公告前的旧趋势，准备接收新信息。

#### ✍️ 输入门（Input Gate）
> "现在有什么新信息值得写进档案？"

$$\boldsymbol{i}_t = \sigma\!\left(W_i \cdot [h_{t-1};\, x_t] + b_i\right)$$

$$\tilde{\boldsymbol{C}}_t = \tanh\!\left(W_C \cdot [h_{t-1};\, x_t] + b_C\right)$$

- $\boldsymbol{i}_t$：决定"写多少"，取值 $(0,1)$
- $\tilde{\boldsymbol{C}}_t$：候选记忆，决定"写什么"，取值 $(-1,1)$
- 两者逐元素相乘 $\boldsymbol{i}_t \odot \tilde{\boldsymbol{C}}_t$ 才是真正写入的量

**股市例子**：RSI 超卖信号出现，输入门决定将"超卖状态"写入记忆。

#### 📤 输出门（Output Gate）
> "档案里哪些内容要拿出来用？"

$$\boldsymbol{o}_t = \sigma\!\left(W_o \cdot [h_{t-1};\, x_t] + b_o\right)$$

决定从细胞状态中读取哪部分作为当前输出 $h_t$，传给 Transformer 和下一时间步。

### 细胞状态与隐藏状态的完整更新公式

**第一步：更新细胞状态**

$$\boldsymbol{C}_t = \boldsymbol{f}_t \odot \boldsymbol{C}_{t-1} \;+\; \boldsymbol{i}_t \odot \tilde{\boldsymbol{C}}_t$$

$$\underbrace{\boldsymbol{f}_t \odot \boldsymbol{C}_{t-1}}_{\text{保留旧档案}} \;+\; \underbrace{\boldsymbol{i}_t \odot \tilde{\boldsymbol{C}}_t}_{\text{写入新信息}}$$

**第二步：计算输出隐藏状态**

$$\boldsymbol{h}_t = \boldsymbol{o}_t \odot \tanh(\boldsymbol{C}_t)$$

这个**加法**结构是关键——梯度可以通过加法直接传回去，不会像普通 RNN 的连乘那样衰减消失。

> **与普通 RNN 对比**  
> 普通 RNN：$h_t = \tanh(W_h h_{t-1} + W_x x_t + b)$  
> 反向传播时梯度正比于 $\prod_{k} W_h^T$，连乘 $T$ 次后 $|W_h|<1$ 则消失，$|W_h|>1$ 则爆炸。  
> LSTM 的 $C_t$ 更新中有加法，梯度路径为 $\frac{\partial C_t}{\partial C_{t-1}} = f_t \approx 1$，远端梯度得以保留。

### 本项目配置

- **4层堆叠**：第1层学低级时序模式，深层学更抽象的趋势
- **hidden=256**：每个时间步的记忆向量是256维
- 输出完整序列 **(64, 256)**，不只取最后一步——因为 Transformer 需要看全部时间步

## 五、Transformer：让时间步互相"投票"

### 直觉类比

假设你要判断今天股价会不会涨，你会这样思考：  
"30天前有个类似的形态，那次之后涨了；15天前出现过放量，那次之后震荡了……"

这就是 **Self-Attention**：让今天直接和30天前、15天前"对话"，不需要靠 LSTM 的逐步传话。

### Q、K、V 三个角色

对每个时间步，把它的向量线性投影成三份：

| 角色 | 类比 | 作用 |
|------|------|------|
| $Q$（Query）| "我想找什么信息？" | 提问者 |
| $K$（Key）| "我能提供什么信息？" | 被检索者的标签 |
| $V$（Value）| "我实际包含的内容" | 被检索者的内容 |

### 缩放点积注意力公式

设输入序列 $X \in \mathbb{R}^{T \times d_{\text{model}}}$（本项目 $T=64,\,d_{\text{model}}=256$）：

**第一步：线性投影**

$$Q = X W_Q, \quad K = X W_K, \quad V = X W_V$$

$$W_Q, W_K, W_V \in \mathbb{R}^{d_{\text{model}} \times d_k}, \quad d_k = \frac{d_{\text{model}}}{\text{nhead}} = \frac{256}{4} = 64$$

**第二步：计算注意力权重并加权求和**

$$\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right) V$$

- $QK^\top \in \mathbb{R}^{T \times T}$：每对时间步之间的相似度（共 $64 \times 64 = 4096$ 个值）
- 除以 $\sqrt{d_k} = \sqrt{64} = 8$：防止点积数值过大导致 softmax 梯度消失
- softmax 沿行归一化：每行加起来 = 1，得到注意力权重矩阵

**第三步：多头拼接**

$$\text{MultiHead}(Q,K,V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)\, W_O$$

$$\text{head}_i = \text{Attention}(XW_{Q_i},\, XW_{K_i},\, XW_{V_i})$$

- $h=4$ 个头，每头输出 64 维，拼接后 $4 \times 64 = 256$ 维，用 $W_O$ 投影回 $d_{\text{model}} = 256$
- 每个头关注不同的时间模式（短期、中期、长期、波动等）

### 位置编码公式

$$\text{PE}(\text{pos}, 2i) = \sin\!\left(\frac{\text{pos}}{10000^{2i/d_{\text{model}}}}\right)$$

$$\text{PE}(\text{pos}, 2i+1) = \cos\!\left(\frac{\text{pos}}{10000^{2i/d_{\text{model}}}}\right)$$

- $\text{pos}$：时间步位置（0 到 $T-1$）
- $i$：维度索引（0 到 $d_{\text{model}}/2 - 1$）
- 偶数维用 sin，奇数维用 cos，使得不同位置的编码向量互不相同且可区分相对距离

**使用方式**：直接加到输入上，$X' = X + \text{PE}$，不引入额外参数。

### Encoder 层完整公式（Pre-LN 变体）

本项目使用 `norm_first=True`（Pre-LN），与原始 Transformer（Post-LN）顺序不同：

**子层一：自注意力**

$$X' = X + \text{MultiHead}\!\left(\text{LN}(X),\, \text{LN}(X),\, \text{LN}(X)\right)$$

**子层二：前馈网络（FFN）**

$$X'' = X' + \text{FFN}\!\left(\text{LN}(X')\right)$$

$$\text{FFN}(z) = W_2 \cdot \text{ReLU}(W_1 z + b_1) + b_2$$

- $W_1 \in \mathbb{R}^{256 \times 1024}$，$W_2 \in \mathbb{R}^{1024 \times 256}$（dim\_feedforward = $4 \times d_{\text{model}}$）
- 两个子层均有**残差连接**（$X +\cdots$），防止梯度消失

> **Pre-LN vs Post-LN 的区别**  
> Post-LN（原论文）：$X' = \text{LN}(X + \text{MultiHead}(\ldots))$，LN 在残差之后  
> Pre-LN（本项目）：$X' = X + \text{MultiHead}(\text{LN}(X))$，LN 在残差之前  
> Pre-LN 的梯度在残差路径上不经过 LN，更容易传播，训练更稳定，无需 warmup。

## 六、三模块如何串联

```
输入 (32, 64, 21)
   ↓
TCN ×4层（膨胀因果卷积）
   ↓ 把21维特征扩展成64维的局部模式
(32, 64, 64)
   ↓
LSTM ×4层（hidden=256）
   ↓ 按时间顺序建立记忆，输出完整序列
(32, 64, 256)
   ↓
位置编码（正弦/余弦，注入时序位置信息）
   ↓
Transformer Encoder ×2层（4头自注意力）
   ↓ 全局梳理，让远近时间步直接关联
(32, 64, 256)
   ↓
取最后时间步 x[:, -1, :]
   ↓
LayerNorm → Linear(256→128)
   ↓
(32, 128) → 送往 H-MoE 融合层
```

**为什么取最后时间步？**  
Transformer 处理完后，最后一个时间步（t=64）的向量，通过 Attention 已经"汇聚"了全部64步的信息，是对整个窗口最全面的摘要。

## 七、情感信号是怎么起作用的

情感特征并不是单独一路，而是**直接拼在数值特征后面**，一起输入 TCN。

```
每天的特征 = [open, close, high, low, ..., ATR, OBV,  |  sentiment_mean, sentiment_std, ...]
               ↑————— 15维技术指标 ——————↑  ↑——————— 7维情感 ————————↑
                                    21维
```

这样，TCN 和 LSTM 在卷积/门控时，会同时看到"今天RSI超卖 + 今天股吧情绪高度悲观"，学到两者联合的信号。

**情感动量**（`sentiment_momentum` = 今天均值 - 昨天均值）比情感绝对值更有预测力——市场更在乎情绪在变好还是变坏。

---

## 八、三句话总结

1. **TCN** 像一个多焦距镜头，同时看到短中长不同尺度的K线形态。
2. **LSTM** 像一个有选择性遗忘能力的记事本，记住真正重要的历史事件。
3. **Transformer** 像一个打了全局电话会议的分析师，让任意两个时间点的信号直接碰头。

三者串联，让模型在预测明天收盘价时，同时具备"识别形态 + 记忆历史 + 全局关联"三种能力。

## 附录：全模型公式速查表

### A. TCN

| 公式 | 说明 |
|------|------|
| $y_t = \sum_{k=0}^{K-1} w_k \cdot x_{t-d\cdot k}$ | 膨胀因果卷积，$d=2^{i-1}$，第 $i$ 层 |
| $\text{RF} = 1 + 2(K-1)(2^L-1)$ | 感受野，$K=3,L=4$ → RF=61 |
| $\text{Block}(\mathbf{u}) = \text{ReLU}(\mathcal{F}(\mathbf{u}) + \mathbf{u}')$ | 残差块，$\mathbf{u}'$ 为 shortcut |
| $(B,T,F) \xrightarrow{\text{TCN}} (B,T,64)$ | 形状变化，21维→64维 |

### B. LSTM

| 公式 | 说明 |
|------|------|
| $\boldsymbol{f}_t = \sigma(W_f[h_{t-1};x_t]+b_f)$ | 遗忘门：决定丢弃多少旧信息 |
| $\boldsymbol{i}_t = \sigma(W_i[h_{t-1};x_t]+b_i)$ | 输入门：决定写入多少新信息 |
| $\tilde{\boldsymbol{C}}_t = \tanh(W_C[h_{t-1};x_t]+b_C)$ | 候选记忆：新信息的内容 |
| $\boldsymbol{C}_t = \boldsymbol{f}_t \odot \boldsymbol{C}_{t-1} + \boldsymbol{i}_t \odot \tilde{\boldsymbol{C}}_t$ | 细胞状态更新（关键：加法结构） |
| $\boldsymbol{o}_t = \sigma(W_o[h_{t-1};x_t]+b_o)$ | 输出门：决定输出细胞状态的哪部分 |
| $\boldsymbol{h}_t = \boldsymbol{o}_t \odot \tanh(\boldsymbol{C}_t)$ | 隐藏状态输出 |
| $(B,T,64) \xrightarrow{\text{LSTM×4}} (B,T,256)$ | 形状变化，64维→256维 |

### C. Transformer

| 公式 | 说明 |
|------|------|
| $Q=XW_Q,\;K=XW_K,\;V=XW_V$ | 线性投影，$d_k=64$/head |
| $\text{Attn}(Q,K,V)=\text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)V$ | 缩放点积注意力 |
| $\text{MultiHead}=\text{Concat}(\text{head}_1,\ldots,\text{head}_4)W_O$ | 多头拼接 |
| $\text{PE}(\text{pos},2i)=\sin\!\left(\frac{\text{pos}}{10000^{2i/d}}\right)$ | 位置编码（偶数维） |
| $\text{PE}(\text{pos},2i+1)=\cos\!\left(\frac{\text{pos}}{10000^{2i/d}}\right)$ | 位置编码（奇数维） |
| $X'=X+\text{MultiHead}(\text{LN}(X))$ | Pre-LN 自注意力子层 |
| $X''=X'+\text{FFN}(\text{LN}(X'))$ | Pre-LN 前馈子层 |
| $\text{FFN}(z)=W_2\cdot\text{ReLU}(W_1 z+b_1)+b_2$ | FFN，256→1024→256 |
| $(B,T,256) \xrightarrow{\text{Attn}} (B,256)$ | 取最后时间步后形状 |

### D. 完整前向传播

$$\hat{y} = f_{\text{proj}}\!\left(\text{LN}\!\left(\text{Transformer}\!\left(\text{PE}\!\left(\text{LSTM}\!\left(\text{TCN}(X)\right)\right)\right)[\,:,\,-1,\,:\,]\right)\right)$$
